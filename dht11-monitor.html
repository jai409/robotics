02 — Connection Guide

Wiring Diagram

Select your sensor type. The module version is the simplest — just three wires.

DHT11 Pin Arduino Pin Wire Colour Notes
VCC → 5 V Red Power supply
GND → GND Black Common ground
DATA → Digital Pin 2 Green / Yellow Serial data signal
ℹ️ The 3-pin module already includes an onboard 10 kΩ pull-up resistor. Simply connect three wires and upload the code — no extra components needed.
DHT11 Pin Arduino / Destination Wire Colour Notes
Pin 1 — VCC → 5 V Red Power supply
Pin 2 — DATA → Digital Pin 2 Green / Yellow Signal line
10 kΩ resistor → DATA ↔ VCC Orange Pull-up — required
Pin 3 — NC — Not connected — Leave floating
Pin 4 — GND → GND Black Common ground
⚠️ The raw 4-pin sensor has no built-in pull-up. Connect a 10 kΩ resistor between Pin 2 (DATA) and Pin 1 (VCC). Without it the data line will float and give unreliable readings.

03 — Firmware

Arduino Code

Full source with sensor validation and three-tier alert logic. Upload directly to your Arduino board.

dht11_monitor.ino 
#include <DHT.h>

#define DHTPIN   2        // Data pin → digital pin 2
#define DHTTYPE  DHT11    // Sensor model

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float humidity    = dht.readHumidity();
  float temperature = dht.readTemperature();

  // Validate sensor readings before processing
  if (isnan(humidity) || isnan(temperature)) {
    Serial.println("Failed to read from DHT sensor!");
    return;
  }

  // Print readings to Serial Monitor
  Serial.print("Humidity: ");
  Serial.print(humidity);
  Serial.print(" %  ");
  Serial.print("Temperature: ");
  Serial.print(temperature);
  Serial.println(" °C");

  // Industrial-style layered threshold alerts
  if (temperature >= 35 && temperature < 45) {
    Serial.println("CAUTION: Moderate temperature");
  } else if (temperature >= 45) {
    Serial.println("CRITICAL: Overheating! Take action!");
  }

  delay(2000);  // Wait 2 s between readings
}

04 — Safety Logic

Alert Thresholds

Three-tier monitoring logic modelled on industrial safety systems. Each level triggers a distinct Serial message.

Level 0 — Normal
T < 35 °C
Readings print normally. No additional alert is generated. System is within safe operating parameters.
Level 1 — Caution
35 °C ≤ T < 45 °C
Moderate temperature detected. Outputs "CAUTION: Moderate temperature" alongside sensor data.
Level 2 — Critical
T ≥ 45 °C
Overheating condition. Outputs "CRITICAL: Overheating! Take action!" — immediate response required.
